# uuptSpecs
UUPT Mobile Service CocoaPods specs.

UU跑腿移动服务iOS CocoaPods仓库。

## 使用
- Podfile中指定仓库位置，
```
http://uugit.uupt.com/zhangwenbin/uuptSpecs.git
```

- 为工程target添加依赖，例：公共地图依赖，

```
pod 'UUPTMapView', '~> 1.0.0'
```
